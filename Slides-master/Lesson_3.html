Publishing your Analysis with Git

  1. knit your document to html (e.g. creates Analysis.html)
  2. Push your html file to Github
  3. On the GitHub Repo go to Settings > Pages
  4. Choose main/master, root hit save
  5. Go to the provided URL, but append the name of your html-file (e.g. Analysis.html)
